### Install project with:
``` bash
$ npm install -g .
```
### and run with:
``` bash
$ users_cli --help
```

Example usage:
``` bash
# adding a user:
$ users_cli add --firstName dimitris --lastName Gkatziouras --academicEmail example@some_mail.com --password 1234
# removing a user:
$ users_cli remove --id 1
```
